# tablered

Десктоп-приложение на Python/tkinter для массового редактирования `.ods`-файлов.

Открываешь один файл-образец, редактируешь значения ячеек и форматирование — и применяешь те же правки ко всем файлам группы одним нажатием кнопки. Файлы группируются по количеству строк с данными на первом листе.

## Запуск

```bash
python3 tableredr.py
```


## Что умеет

- Сканирует папку и группирует `.ods`-файлы по количеству строк
- Встроенный редактор таблицы с inline-редактированием ячеек
- Форматирование текста: шрифт, размер, жирный, курсив, цвет
- Сохранение одного файла или массовое применение ко всей группе
- Прогресс-бар при массовом применении
- Лог ошибок в интерфейсе
